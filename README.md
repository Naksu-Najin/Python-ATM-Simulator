# Python ATM Simulator

A simple Python program that simulates an ATM with PIN verification, balance checking, deposits, and withdrawals.

## Features
- PIN authentication (3 attempts allowed)
- Check balance
- Deposit money
- Withdraw money (with insufficient balance check)
- Input validation for numbers and positive amounts

## How to Use
1. Run `atm.py` in a Python environment (Python 3.x recommended).
2. Enter your PIN when prompted.
3. Choose an option from the menu:
   - `1` - Check Balance
   - `2` - Deposit
   - `3` - Withdraw
   - `4` - Exit
